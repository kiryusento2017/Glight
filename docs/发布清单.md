# 发布清单（防报毒 SOP）

> 目标：纯免费档，把无签名 Go exe 从 GitHub 下载时的误报降到最低。
> 定稿 2026-06-06。约束：作者为中国个人开发者，签名根治不可行（见末尾「备查」），故走免费组合拳 + 长期信誉累积。

## 每次发版固定走（按顺序勾）

- [ ] **1. 发布构建**：`go build -trimpath -buildvcs=false -ldflags="-H windowsgui" -o dist/claude-traffic-light.exe .`（发行版 exe 一律输出到 `dist/`，与源码隔离；dist 不存在先建）
  - ✅ **必带 `-trimpath -buildvcs=false`**——清除编译机绝对路径（GOROOT、含 Windows 用户名的依赖 cache 路径、项目源码路径）+ git 元数据，否则会泄露作者本机信息。详见 CLAUDE.md「构建规则」。
  - ⚠️ **绝不加 `-s -w`**——strip 符号会直接触发/加重 Go 二进制的 `Trojan:Win32/Wacatac.!ml` 误报。这是已查证的坑，保持现状别踩。
  - 验证清零：`grep -a -c "你的用户名" claude-traffic-light.exe` 应为 0。
- [ ] **2. 不加壳**：不用 UPX 等压缩壳（壳是误报头号特征）。
- [ ] **3. 自检**：把 exe 传 [VirusTotal](https://www.virustotal.com)，记录**哪些引擎报、报什么名**（留作申诉证据 + 心里有数）。
- [ ] **4. 提交误报申诉**（免费，对 Defender 最有效）：
  - Defender → https://www.microsoft.com/en-us/wdsi/filesubmission
  - SmartScreen → https://feedback.smartscreen.microsoft.com
  - VirusTotal 里其余报的引擎，逐个走它们的 false-positive 提交入口
  - ⚠️ 申诉是**针对具体文件哈希**的——**每个新版本 exe 哈希都变了，必须重新提交**。
- [ ] **5. Release 页固定附**：
  - `SHA256` 校验值
  - 一句话说明 + 截图：「这是无签名 Go 程序的已知误报，点 More info → Run anyway，或加 Defender 排除」
  - VirusTotal 扫描结果链接（自证清白）

## 长期

下载量上去后，SmartScreen 信誉自然累积，警告逐步消退。免费档的本质是「把误报压到最低 + 撑过信誉冷启动期」。

## 升级项（按需，非必需）

若免费档做扎实后仍嫌每次申诉烦，可花小钱上 **Certum 个人级开源代码签名证书**（欧洲 CA，对个人/开源最便宜）。效果：去掉「未知发布者」标签 + 给信誉累积一个稳定载体 + 降部分杀软启发式分。**但仍非即时根治**，且每年续费。

## 备查：为什么签名根治这条路不走

| 方案 | 成本 | 能否用 | 即时见效 |
|---|---|---|---|
| Azure Artifact Signing | $9.99/月 | ❌ 个人版只开放美/加，中国个人不符资格 | — |
| EV 证书 | $250–700/年 | ❌ 只发注册企业 | ❌ 自 2024 起 EV 也不再即时给 SmartScreen 信誉 |
| OV/IV 证书（Certum/Sectigo） | ~€30–$225/年 | ✅ 个人可申 | ❌ 新证书从零信誉，初期照样可能警告 |

**关键认知**：自 2024 微软改规则后，花钱买证书也换不来「即时干净」，OV/EV 都得靠下载量累积信誉。所以不值得为「立刻根治」买证书——买了也不立刻根治。

## 误报根因（理解用，不影响操作）

报毒有两层独立来源，免费档同时打这两层：
- **SmartScreen**：基于「签名 + 下载信誉」，无签名 + 新文件 → 零信誉拦截。靠申诉 + 信誉累积消退。
- **Defender/杀软启发式（Wacatac.!ml）**：ML 按二进制特征猜，Go 二进制是重灾区，叠加本程序的高危画像（抓屏 + 枚举进程 + 写注册表自启 + 改 settings.json）。靠构建卫生（不 strip、不加壳）+ 申诉缓解。
